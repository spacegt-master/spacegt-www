# 🇯🇵 日语自学笔记：工程师的系统化学习

对我而言，学习日语不仅仅是为了兴趣，更是一次**系统重构**——去理解一种全新的语言逻辑和文化架构。我的目标是：**高效、精准、可维护**地掌握这门语言，最终达成无字幕观看高分日系动漫的成就！

## ⚙️ 学习蓝图：将日语视为一个系统

我将日语学习过程分解为以下几个核心模块，确保学习的结构化和可追踪性：

### 1. 底层协议：五十音图 (Kana — Byte Level)

* **定位：** 相当于网络通信中的**底层编码**，必须做到反射式、零延迟识别。

* **状态：** 已完成掌握，正在通过阅读和听力进行**强化与固化**。

* **目标：** 将平假名和片假名的识别速率提升至母语水平。

### 2. 语法结构：文法 (Grammar — The Compiler)

* **定位：** 系统的**语法规则和运行逻辑**。理解了文法，才能将数据包（单词）按照正确的顺序进行组织和编译。

* **当前重点：** 正在攻克 **N5/N4** 阶段的句型和格助词的细微区别。

* **难点聚焦：** **助词** (`は`、`が`、`を`、`に`) 的精确使用，它们是语句含义的**路由选择**。

### 3. 数据层：语汇 (Vocabulary — Data Structures)

* **定位：** 语言的**数据结构和 API 接口**。记忆单词，就是丰富系统的内存和功能。

* **方法：** 使用 **间隔重复（Anki）算法**进行高效记忆，确保在最佳时间点进行回顾，防止遗忘曲线劣化。

* **结合自身：** 特别学习与**健康、作息**相关的词汇（例如：睡眠、骑行、正畸），将学习内容与日常生活强绑定。

### 🎯 学习策略（工程师方法论）

1. **最小化切换成本：** 尽量在固定的时间段进行学习，避免打乱我的**高效作息计划**（11点前睡觉，7点起床）。

2. **错误追踪：** 像处理 Bug 一样对待语法错误，找到错误的**根本原因（Root Cause）**并记录在案。
